# To-Do List em C

Projeto simples de lista de tarefas desenvolvido em linguagem C para praticar lógica de programação e meus conhecimentos práticos.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnólogias utilizadas
- Linguagem C
- Git
- GitHub

## Como compilar
No terminal execute:
clang main.c -o todo

## Como executar

Depois de compilar, execute:
./todo

## Objetivo
Projeto desenvolvido para aprendizado e montagem de repertório no Github.
